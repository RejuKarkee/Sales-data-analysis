# Superstore Sales Data Analysis (Python + Excel)
This project analyzes the Superstore dataset using Python, pandas, and Excel.
It includes data cleaning, exploratory analysis, visualizations, and business insights to understand trends across months, regions, categories, and customer segments.

# Key Insights

* Sales peak in February and August
* West region generates the highest total sales
* Technology category performs best
* Consumer segment contributes the largest revenue share
* Top-selling products belong mainly to Tech and Furniture

# Tools Used

* Python (pandas, matplotlib)
* Jupyter Notebook
* Excel (Pivot Tables + Dashboard)
